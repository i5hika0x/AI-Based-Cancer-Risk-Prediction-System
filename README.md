AI-Based Cancer Detection System
Overview
This project is a machine learning-based web application designed to analyze tumor characteristics and predict cancer risk. The system classifies tumors as benign or malignant using structured clinical input features and provides probability-based results for better interpretation.
Features
* Machine learning-based tumor classification
* Probability-based prediction output
* Manual input and CSV file upload support
* Interactive dashboard with data visualization
* Multi-cancer selection interface (Breast, Lung, Oesophagus)
* Integrated search for nearby oncology specialists
Technologies Used
* Python
* Scikit-learn
* Streamlit
* NumPy
* Pandas
* Matplotlib
Deployment
The application is deployed using Streamlit Community Cloud and can be accessed via a public web link.
